# 安全策略

## 支持版本

| 版本   | 支持状态       |
|--------|--------------|
| 1.2.x  | ✅ 活跃支持   |
| < 1.2  | ❌ 不再支持   |

## 报告漏洞

如果你发现安全漏洞，请**不要**在公开 Issues 中报告。

请发送邮件至：**security@jishics.com**

我们会在 48 小时内确认收到，并在 7 天内提供修复方案。

## 安全设计

### 数据保护
- MCP Server **仅暴露公开信息**（服务商名称、价格、评分）
- 客户联系方式、财务数据等 **PII 永不返回**
- 敏感数据在 API 层已过滤，不会进入 MCP 传输

### 接口安全
- 支持 API Key 认证（可选）
- 内置频率限制（Rate Limiting）
- 所有工具调用经过数据过滤器

### 依赖安全
- 定期审查 `npm audit` 结果
- 依赖锁定在 `package-lock.json`
